Color Theory: An Overview

A comprehensive guide to how color systems work and why they matter in clinical and digital environments.

The Color Wheel

The color wheel is the basis of color theory. It consists of Primary, Secondary, and Tertiary colors. Understanding their relationship helps in creating balanced designs.

Color Schemes

Using the color wheel, we can create specific harmonies:

  • Monochromatic: Variations of a single hue.
  • Analogous: Colors located next to each other.
  • Complementary: Colors opposite each other (high contrast).

Additive vs. Subtractive Color

In web design, we use the RGB (Additive) model, whereas in print (like my magazine project), we use CMYK (Subtractive).
